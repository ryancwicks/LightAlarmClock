Light Alarm Clock Project - E-Ink Display

An E-Ink display was chosen for the alarm clock because the display does not produce any light of it's own. I don't know how this affects sleep hygiene, but if you have ever not been able to sleep, the LED time shining in your face isn't any help, so a time display that only turns on when requested makes sense to me.

I chose a 4.2 " e-Paper Module from Waveshare (after watching a video from Great Scott talking about how to interface with these modules:
