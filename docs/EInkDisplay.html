Light Alarm Clock Project - E-Ink Display

An E-Ink display was chosen for the alarm clock because the display does not produce any light of it's own. I don't know how this affects sleep hygiene, but if you have ever not been able to sleep, the LED time shining in your face isn't any help, so a time display that only turns on when requested makes sense to me.

I chose a 4.2 " e-Paper Module from Waveshare (after watching a video from Great Scott talking about how to interface with these modules:

To test the board upon first reciept, I used the Arduino library manager to download the GxEPD2 library. Looking at the example Gx_EPD2.ino, there is a suggested wiring for the ESP32 boards:

// mapping suggestion for ESP32, e.g. LOLIN32, see .../variants/.../pins_arduino.h for your board // NOTE: there are variants with different pins for SPI ! CHECK SPI PINS OF YOUR BOARD // BUSY -> 4, RST -> 16, DC -> 17, CS -> SS(5), CLK -> SCK(18), DIN -> MOSI(23), GND -> GND, 3.3V -> 3.3V

Further down in the example, under the ESP32 ifdef, I uncommented the line:

GxEPD2_BW display(GxEPD2_420(/*CS=5*/ SS, /*DC=*/ 17, /*RST=*/ 16, /*BUSY=*/ 4));
