Light Alarm Clock Project - Firmware

There are several different components to the light alarm clock firmware:

  • Web Server
    • WiFi Hotspot (setting up network).
    • Web Server serving configuration pages.
  • RTC/Alarm Control
  • Light Control
  • File System
    • Save wifi info persistently
    • Save static web pages
    • Load static web pages onto device over serial

File System

I'll use the FFAT file system to store web pages and persistent data on the EEPROM. Be sure to set the partition scheme to "Default with FFat" in the tools menu, or you won't be able to load the file system. If you don't have this option in the tools menu (the DOIT boards don't), you'll have to manually backup the defaults.csv and copy over it using the defaults_ffat.csv file found in your Arduino settigs under ~/.arduino15/packages/esp32/hardware/esp32/1.0.2/tools/partitions (on linux, anyway).
